An empty fill up component

The empty fill up component is a simple way of portraying emptiness: An empty cart, a 404 page, e.t.c.

Passing a 'headerText input to the empty fill up component
[headerText]="'This is a headerText'"
[headerText]="'Your cart is empty'"
Passing a paragraph text to the empty fill up component
[text]="'we've got no cookies for you but some will be up for grabs tomorrow'"
[text]="'Your cart is currently empty but you can start filling it up here:'"
Css variables for customizing the empty fill up component
  • --width: customizes the width of the whole component.
  • --image-width: Customizes the width of the empty fill up image.
  • --header-text-color: customizes the color of the header text
  • --text-color: customizes the color of the paragraph text
  • --text-background: customizes the background color of the empty fill up component
