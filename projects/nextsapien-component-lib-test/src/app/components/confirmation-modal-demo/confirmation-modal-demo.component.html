MultiField
MultiField is an editable list component where each row can consist of one or more fields of various types.
Properties
Type Name Description Data type Required Default Demo
Input title The text that is displayed at the top of the modal. string false 'Are you sure you want to enable demo?'
Input dialogMessage The text that is displayed below the title. string false 'This is the demo dialog message.'
Input confirmText The text that is displayed on the confirm modal botton. string false 'Save'
Input warningModal Determines the theme of the modal. boolean false false
Input hideCancelAction Hide the cancel button on the confirmation modal dialog. boolean false false
