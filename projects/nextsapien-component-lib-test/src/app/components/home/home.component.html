The Next Sapien Component Library provides reusable, easily configurable Angular components to be consumed across Cosential projects in order to provide consistent look and feel, functionality, and implementation.
Requirements
  • Many Next Sapien components wrap or otherwise consume Angular Material components, and as such, any projects consuming this library must first install the requisite Angular Material npm packages. Please see the Angular Material documentation for installation instructions.
  • To make the Angular Material components integrate with the Next Sapien design language, there are recommended CSS overrides which should be included (possibly with modification as necessary) in the consuming project. The relevant CSS overrides are noted in the detail for each Cosential component.
Installation
Run npm run package to build the component library and package it into a tarball. The tarball will be created in the dist folder, from which it can be copied into other angular projects. Install the library using npm install /path/to/tarball
